# Object-DetectionUsingYOLOv3

## Technologies

- Alturos.Yolo version 3.0.6-alpha
- Alturos.YoloV2TinyVocData.1.0.0 
- OpenCvSharp3-AnyCPU ver 4.0.0 stable
